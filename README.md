# tooth-and-pail
crowd sourced tooth fixes

step 1:

run npm install

step 2:

run npm start

step 3:

open index.html in a web browser

step 4:

REACT QUICKLY!

to run tests:  npm run test

To CREATE database  - install KNEX globally
                    - in the terminal run knex migrate:latest
                    -                 run knex seed:run    

To run the db server - in /dbserver
                        nodemon dbserver.js      - will run on localhost:3000

To run the selenium server manually     java -jar selenium-server-standalone-2.45.0.jar -port 4444 -host 127.0.0.1
